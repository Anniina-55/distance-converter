# Distance converter

This Android app is implemented in Kotlin language using Jetpack Compose.  
It converts miles to kilometers (1 mile = 1.609 km) with simple user interface.

---
## Screenshot

![App screenshot](distance-converter.png)

## Features

- Input field for miles 
- State management for user input
- Conversion calculation for positive numbers
- Strings stored in `strings.xml` resource file
- Shows an error message if the user enters a negative value
